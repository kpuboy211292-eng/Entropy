# Entropy
Click the button to start mining NACKL. The entropy miner collects hardware entropy while you use your PC/Mac. Listen to music, type documents, watch movies, and surf the internet. The more you use your PC/Mac, the greater the reward.



Click the button to start mining NACKL. The entropy miner collects hardware entropy while you use your PC/Mac. Listen to music, type documents, watch movies, and surf the internet. The more you use your PC/Mac, the greater the reward.
In order to use sound mining, you need to connect a microphone.


Нажмите кнопку, чтобы начать майнинг NACKL. Entropy майнер собирает аппаратную энтропию, пока вы используете свой ПК/Mac. Слушайте музыку, печатайте документы, смотрите фильмы и пользуйтесь интернетом. Чем больше вы используете свой ПК/Mac, тем больше вознаграждение.
Для того чтобы использовать майнинг с помощью звука требуется подключить микрофон


For Mac users : Mining on a Mac requires permission for Accessibility (required for capturing input sources), Screen and System Sound Recording, and Location Services. The program should prompt you to add these permissions when you first launch it.
                Для майнинга на Mac требуется дать разрешение на Универсальный доступ (нужно для захвата источников ввода), Запись экрана и системного звука, Службы геолокации. При первом запуске программа сама должна попросить добавить разрешения

For Win userd: Mining on Windows requires .NET 8.0 Desktop Runtime and WebView2 Runtime to be installed.
               Для майнинга на Windows требуются установленые .NET 8.0 Desktop Runtime и WebView2 Runtime 





1. .NET 8.0 Desktop Runtime: Download link (Select the "Desktop Runtime" version -> your platform). https://dotnet.microsoft.com/en-us/download/dotnet/8.0
2. WebView2 Runtime: Download Evergreen Bootstrapper. https://developer.microsoft.com/en-us/microsoft-edge/webview2/?form=MA13LH#download
